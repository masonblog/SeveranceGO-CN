# SeveranceGO-CN | 中国离职补偿金微信小程序

![WeChat Mini Program](https://img.shields.io/badge/WeChat-Mini%20Program-07c160?logo=wechat&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-5-3178c6?logo=typescript&logoColor=white)

SeveranceGO-CN 是一个原生微信小程序，用于估算中国大陆劳动关系解除、终止相关的一次性金额。它覆盖常见场景：经济补偿 N、违法解除/终止 2N、第 40 条代通知金，以及未签书面劳动合同二倍工资差额。

当前小程序版只做本地计算：不联网、不收集联系方式、不提交咨询线索。

## 功能

- 原生小程序表单，适配手机端填写。
- 内置 2024 年平均工资和最新最低工资参考数据，并展示来源、口径和数据等级。
- 本地展示估算区间、工作年限、补偿月数、金额明细、适用规则、风险提示和建议核查事项。
- 保留纯 TypeScript 计算、校验和地区数据测试，方便复核高风险法律/数据逻辑。

## 本地开发

```bash
corepack pnpm install
corepack pnpm test
corepack pnpm build
```

用微信开发者工具打开项目根目录。`project.config.json` 当前使用 `touristappid`，上线前请替换为真实小程序 AppID，并按微信公众平台要求完成类目、备案和审核配置。

## 表单填写

1. 填写实际入职时间、离职或预计离职时间、上一自然年度总收入。
2. 选择劳动合同履行地，核对系统带出的平均工资和最低工资口径。
3. 选择是否已签书面劳动合同；已签时填写合同签订时间。
4. 选择离职情况；如果涉及被迫离职、第 40 条未提前通知等情况，补充对应选项。
5. 点击“本地计算”，查看估算结果。

## 数据口径

地区工资数据说明见 `docs/data-sources.md`。

对于未公开统一城市级 2024 平均工资的城市，项目按省级官方、省内片区官方或国家统计局 2024 分区域口径降级预填，并在页面提示数据等级；最低工资默认使用当前已核验的人社部全国表、地方公告中的城市/区县档位，或省级第一档参考值。

## 法律提示

本工具只用于初步估算，不构成法律意见。地区工资数据、司法口径、解除理由、证据材料和程序事实都会影响最终结果，正式使用前应复核城市数据与最新法规。
